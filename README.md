# Keyboard Events Test Page

This page allows you to test keyboard events in your browser.

https://tats-u.github.io/kbdevents

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses the Next.js App Router and exports static files to `out/` when you run `yarn build`.
